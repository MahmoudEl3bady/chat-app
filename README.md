# React Firebase Chat App

A real-time chat application built with React,TypeScript,TailwindCSS and Firebase.

## Features

- Real-time messaging
- User authentication
- Create and delete chats
- Clear chat history
- Responsive design

## Technologies

- React
- TypeScript
- Firebase (Firestore, Authentication)
- TailwindCSS
- Chakra UI
- React Query
- React Router
- Date-fns
- React Icons
