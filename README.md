# React Firebase Chat App

A real-time chat application built with React,TypeScript and Firebase.

## Features

- Real-time messaging
- User authentication
- Create and delete chats
- Clear chat history
- Responsive design
- Search functionality

## Technologies

- React
- TypeScript
- Firebase (Firestore, Authentication)
- Chakra UI
- React Query
- React Router
- Date-fns
- React Icons
